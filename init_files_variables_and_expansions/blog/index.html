Understanding the syntax

The syntax is composed of the command ls which means list. This will give you a list of all the content of a file or directory. Also it contains the wildcard * which means you the user want the shell to look for any file containing the characters you specified like in this case files that contain ".c". The .c stands for files that are written in c language which is a language used in proggraming to help the computer process and display the information which is contained in the file. In other words you want shell to look for all files written in c language and list them.
